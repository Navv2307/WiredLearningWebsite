Data Analytics with Python and SQL


Course Description:

This course provides a robust foundation in the fundamental tools for data analysis and manipulation. You'll develop core competencies in database querying with SQL, data wrangling and analysis with Python, and draw actionable insights from data.


Prerequisites:

  • A Laptop or Desktop with a Stable Internet Connection.
  • Some familiarity with programming concepts would be helpful but not essential, as we will go through everything from scratch.

Course Modules:


Module 1: Introduction to Data Analysis

  • Defining data analysis, its goals, and processes
  • Types of data (structured, semi-structured, unstructured)
  • Key data analysis tools and techniques
  • Types of Charts and Its Use cases?
  • What is Data Cleaning?
  • What is Data Modelling and Its Types?
  • Types of Data Analysis Techniques
  • Data Integrity, Data Governance
  • Extract, Transform, and Load (ETL) Explained
  • Complete Life Cycle of a Data Analysis Project

Module 2: Statistics and Probability Theory

  • Basic Statistical Concepts:
    • Measures of central tendency (mean, median, mode)
    • Measures of dispersion (range, variance, standard deviation)
    • Probability distributions (normal, binomial, Poisson)
    • Statistical inference and hypothesis testing
  • Introduction to Probability Theory:
    • Basic probability rules and concepts
    • Probability distributions and their applications
  • Exploratory Data Analysis (EDA) Techniques:
    • Histograms, box plots, and scatter plots
    • Inter-Quartile Range, Identifying outliers and distributions of data

Module 3: SQL Fundamentals

  • What is a Database?
  • Database Management Systems (DBMS) and Relational Database Management Systems (RDBMS)
  • Explaining the database Schema
  • What is Data Modelling and Types of Data Modelling, Explaining ER-Diagram
  • SQL Components (DDL, DML, DCL, TCL)
  • Aggregate functions (SUM, AVG, COUNT, MIN, MAX)

Module 4: Advanced SQL

  • Joining tables (INNER JOIN, OUTER JOIN, LEFT JOIN, RIGHT JOIN)
  • Subqueries and Nested Queries
  • Views, Stored Procedures, and Triggers
  • Window functions

Module 5: Fundamentals of Python

  • Introduction to Python
  • Data types (numbers, strings, lists, tuples, dictionaries)
  • Variables, operators, and control flow (if/else, loops)
  • Functions and modules
  • File Handling
  • List Comprehension, Functional Programming
  • Error Handling
  • Basics of OOP

Module 6: Essential Python Libraries

  • NumPy: Introduction to Numpy, Arrays and Numerical Operations
  • Pandas: DataFrames, Series, data manipulation, data cleaning and data processing, pandas profiling
  • Matplotlib / Seaborn: Data visualization techniques (line plots, scatter plots, histograms, bar charts, etc.)

Module 8: Data Storytelling & Visualization

  • Principles of effective data visualization
  • Choosing the right chart types for different insights
  • Creating compelling dashboards and reports

Module 9: Creating Dashboards with Looker Studio

  • Introduction to Looker Studio and its capabilities
  • Connecting to data sources and building queries
  • Designing and customizing dashboards in Looker Studio
  • Implementing interactivity and filters
  • Sharing and collaboration features in Looker Studio

> Assessments and Quizzes and Others

  • Weekly Assessments/Quizzes to ensure that your knowledge is on par with the course.
  • Peer Reviews
  • Case Study reviews

> Projects with Real World Datasets

  • Build a Database Schema for a School System.
  • Data Processing and Cleaning with Pandas
  • Exploratory Data analysis with Python
  • Build a Dashboard In Looker Studio

> Tools and Technologies 

  • VS Code, MySQL, Jupyter Notebook, Looker Studio, etc.
  • Python Libraries (Numpy, Pandas, Seaborn, Matplotlib)

Note: Throughout the course, we will guide you through the installation process for all necessary software and tools required, including Python, MySQL, Jupyter Notebook, and Looker Studio. You don't need to have them pre-installed, as we'll cover the setup step-by-step during the course sessions. This ensures that everyone can follow along seamlessly regardless of their prior experience or setup.
